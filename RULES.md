# Terroir HUB WINE — 制作ルール

## 1. 情報の正確性（最重要）

### 嘘をつかない原則
- **ワイナリーが公式に出している情報だけを使う**
- 情報がない項目は「表示しない」。推測で埋めない
- AIが勝手に文章を生成・補完しない

### 引用ルール
- ワイナリーサイト・公式SNSからの文章引用には**必ず出典を明記**する
  ```html
  <div style="font-size:10px;color:var(--text-muted);margin-top:10px;">
    出典：<a href="https://example.com" target="_blank" rel="noopener">ワイナリー名 公式サイト</a>
  </div>
  ```
- 醸造家・オーナーの個人名での発言は、**その人物が実際にその発言をしたと確認できる場合のみ**帰属する
- 確認できない場合はワイナリーの一般的な説明として記述する

### データソースの優先順位
1. ワイナリー公式サイト（最優先）
2. ラベル法定表記（アルコール度数・原材料・原産地等）
3. 公式コンペティション結果（日本ワインコンクール / SAKURA AWARDS / IWC）
4. 国税庁オープンデータ（ワイナリー名・住所・免許情報）
5. 各都道府県ワイン組合・産地協議会の公式情報
6. ワイナリーのSNS公式アカウント

### 使ってはいけないソース
- 個人ブログ・口コミサイトの主観的評価
- AIによる推定・生成テキスト（ファクトチェック不可のもの）
- 出典不明の二次情報

## 2. 銘柄カードの情報設計

### 必須項目（ラベル・公式サイトから取得可能）
- ワイン名
- 種類（赤ワイン・白ワイン・ロゼ・スパークリング・甘口等）
- アルコール度数
- 使用品種
- 説明文（公式サイトの記述ベース）

### 条件付き表示項目（データがある場合のみ表示）
- **GI認定**：GI山梨・GI北海道・GI長野・GI山形等
  - 認定を受けているワイナリーのみ表示
- **ヴィンテージ**：収穫年
  - 公式に記載がある場合のみ
- **熟成**：樽熟成・シュール・リー等の情報
  - ワイナリーが公開している情報のみ
- **受賞歴**：年・コンペ名・賞名
  - 公式発表で確認できるもののみ
  - 受賞歴がない銘柄にはセクション自体を表示しない

### 日本ワインの定義（重要）
- **日本ワイン** = 国産ぶどうを100%使用し、日本国内で醸造したもの
- 輸入濃縮果汁や輸入ワインを使用したものは「国内製造ワイン」であり「日本ワイン」とは区別する
- 本サイトは**日本ワインのみ**を対象とする

### 画像
- 正方形（aspect-ratio: 1/1）のプレースホルダーを標準装備
- ワイナリーから使用許可を得た写真 → `<img class="brand-img">` に差替
- 未許可のワイナリー → プレースホルダー表示（`brand-img-placeholder`）
- Unsplash等の汎用画像は**ワイナリーや銘柄の固有ページには使わない**（誤解を招く）
- ポータルトップの一般的なセクション（地域紹介等）ではUnsplash使用可

## 3. GI（地理的表示）の扱い

### 基本方針
- GIは国税庁が認定する地理的表示保護制度
- GI認定を受けたワイナリーは、要件を満たすワインにGI表示が可能
- サイトではGI認定の有無を明確に区別する

### GI認定産地（2026年4月現在）
| GI名 | 対象地域 | 認定年 |
|------|---------|--------|
| GI山梨 | 山梨県 | 2013年 |
| GI北海道 | 北海道 | 2018年 |
| GI長野 | 長野県 | 2020年 |
| GI山形 | 山形県 | 2023年 |
| GI大阪 | 大阪府 | 2024年 |

### GI表示ルール
- GIバッジは**実際にGI認定産地のワインを造るワイナリー**にのみ表示
- GI認定を申請中・検討中のワイナリーには表示しない
- ワイナリーの所在地ではなく、**ワインに使用するぶどうの産地**がGI要件を満たす必要がある

## 6. SEO・AIO戦略

### 構造化データ（必須）
- 全ワイナリー：LocalBusiness（additionalType: Winery）+ Product schema
- 教科書ページ：FAQPage schema
- テイスティングガイド：HowTo schema
- パンくずリスト：BreadcrumbList schema
- サイト検索：SearchAction schema

### AIO（AI Overview）対策
- 教科書ページは「問い→答え」形式で構成
- 1段落200文字以内で結論を先に述べる
- 英語版を必ず用意（英語圏のAI Overviewに引用されるため）
